# Survivor Island

**Survivor Island** est un jeu de survie développé sous **Unity (PC uniquement)**, en vue **FPS / troisième personne**, dans un environnement de type île, inspiré de jeux comme *The Forest*.

Ce projet est réalisé dans un **cadre scolaire**, avec pour objectif de concevoir un jeu fonctionnel, structuré et évolutif, en appliquant de bonnes pratiques de développement.

---

## Concept du jeu

Le joueur est abandonné sur une île et doit survivre en :
- explorant l’environnement,
- gérant ses besoins vitaux,
- collectant des ressources,
- fabriquant des objets,
- se défendant contre des ennemis.

---

## Fonctionnalités principales

- Déplacement du joueur (FPS / caméra libre)
- Gestion des besoins vitaux :
  - Santé
  - Faim
  - Soif
  - Fatigue
- Système d’inventaire
- Collecte de ressources (arbres, rochers, objets)
- Interactions via raycast (`E` pour interagir)
- Interface utilisateur (HUD)
- Environnement insulaire
- Compatible **PC uniquement**

---
